# Projekt M295
um diesen Projekt zu starten muss man Express package installieren mit dem command : ```npm i express```<br>
Damit die Session funktioniert, muss man zusätzlich noch den Express-Session package herunterladen: ```npm i express-session```<br>
Den body-parser installiert man anhand von ```npm i body-parser```<br>
Um diesen Projekt zu starten muss man nur ```node main.js``` ausführen
